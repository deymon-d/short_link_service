# Short link service

## Сборка
```bash
mvn clean compile
mvn -B package -DskipTests
```

После чего будет получен .jar файл, который уже можно запустить.

## Поддерживаемые команды

Все команды исполняются вводом номера предложенной команды.

### Команды без входа в аккаунт
1. login: используя логин и пароль позволяет авторизоваться.
2. register: позволяет создать новый аккаунт, сохраняя хэш пароля. 
3. redirect: позволяет разыменовать короткую ссылку (по возможности, автоматом открыть в браузере)
4. exit: завершить исполнение.

### Команды внутри аккаунта
1. create short link: позволяет создать новую короткую ссылку, можно указать описание, название, а также временные и количественные лимиты.
2. redirect: работает аналогично редиректу без входа в аккаунт. 
3. list my links: позволяет посмотреть список ваших ссылок.
4. update link: позволяет обновить ограничения или описания короткой ссылки, требует, чтобы вы были авторизованы и автором данной ссылки.
5. delete link: позволяет удалить созданную ранее вами ссылку по короткому имени.
6. logout: позволяет сменить аккаунт.
7. help: выводит подробное описание команд.
8. exit: завершить исполнение.

## Тесты

Для тестов можно запустить следующую команду:
```bash
mvn clean verify
```
